# Admin Dashboard

Interface d'administration pour les marchands OmniSphere.

## Fonctionnalités

- CRM pour la gestion des clients
- Gestion des commandes
- Gestion des produits
- Tableau de bord avec statistiques

## Développement

```bash
# Installer les dépendances
npm install

# Lancer en mode développement
npm run dev

# Builder pour la production
npm run build

# Lancer en production
npm start
```

L'application sera accessible sur http://localhost:3000

## Structure

- `pages/dashboard.tsx` - Page principale du dashboard
- `styles/theme.ts` - Configuration du thème
- `components/` - Composants réutilisables (à créer)
