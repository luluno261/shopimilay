# Storefront Template

Template de base pour la boutique cliente OmniSphere (SSR/SSG).

## Fonctionnalités

- Template de boutique personnalisable
- Builder de sections de boutique
- SSR/SSG avec Next.js

## Développement

```bash
npm install
npm run dev
```

L'application sera accessible sur http://localhost:3000

## Structure

- `pages/index.tsx` - Page d'accueil
- `components/builder/` - Composants du builder de boutique
